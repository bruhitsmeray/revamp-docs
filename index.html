The following sections of the documentation represents the Aleph framework and it's code as they function on an upcoming build based on Unreal Engine 4.27.2!

Aleph is a former learning project, that now grows up to be a framework for my future game projects. I intend to share this base with you, so I find it necessary to leave you some documentation too.
